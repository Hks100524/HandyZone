# 🛒 HandyZone – Diploma Project

HandyZone is a modern and responsive **Frontend E-Commerce Web Application** developed as a **Diploma Project** using **HTML, CSS, and JavaScript**. The project is designed to provide a smooth and interactive online shopping experience with clean UI, responsive layouts, and user-friendly navigation.

This project demonstrates practical knowledge of frontend web development, UI/UX design principles, responsive website structure, and multi-page website implementation.
